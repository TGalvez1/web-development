The sole purpose of this website is to help people who want to learn how to type at a much faster rate than any other website like this can.

How will this achieve that goal?

By providing specific assistance to each user, unique to them. Everyone uses the keyboard differently, everyone learns differently. I want to be able provide a place where people can learn typing in a different way and specific to their needs by using these various metrics.
