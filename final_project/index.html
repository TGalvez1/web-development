Welcome to Analytical Typing! The best way to learn how to type.

Analytical Typing allows you to see how you type on the keyboard. Analytical Typing provides an in-depth look into many stats of your typing such as: typing speed (wpm), what letters do you most commonly mix up, at what point do you have to stop to keep typing, etc.

For more info, head over to the Help page! The Help page will tell you all about the various metrics the game records to provide you the best feedback.

If you would like to learn more about the purpose behind the webpage and also more about the creator, head over to the About page!
